PhotoArrow v1.0-beta - Strzałki dla kierunku zdjęć
Data publikacji: 22 sierpnia 2016

Nowe funkcje

Użycie
PhotoArrow start {centymetry}

#Uruchomienie aplikacji
mdl load PhotoArrow
#Podniesienie wysokości o 10cm
PhotoArrow start 10
#Obniżenie wysokości o 1m
PhotoArrow start -100

** Zmiany (historia)
Do zrobienia
* podręcznik użytkownika
* sprawdzenie, czy jest podłączony plik referencyjny z kierunkami zdjęć
* wczytanie zdjęć zrobionych dla gospodarstwa z katalogu DOKUMENTACJA_FOTOGRAFICZNA
* wstawienie strzałek do pliku dla zrobionych zdjęć
* raport z podsumowaniem do pliku
